# Client (Next.js)

Frontend de FIKHI CONSTRUCTION.

## Lancer en dev

```bash
npm run dev
```

Application disponible sur `http://localhost:3000`.

## Variables d environnement

Creer `client/.env.local`:

```bash
NEXT_PUBLIC_API_URL=http://localhost:5000/api
```

## Scripts

```bash
npm run dev
npm run lint
npm run build
npm run start
```
